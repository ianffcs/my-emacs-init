#+TITLE: Keybindings Reference
#+AUTHOR: Ian Fernandez
#+STARTUP: overview

All custom keybindings use the =C-c= prefix. Each prefix maps to a
transient menu — press the prefix alone to open the menu, or add a
suffix directly for the shortcut.

* Prefix Map

| Prefix  | Domain               | Menu function          |
|---------+----------------------+------------------------|
| =C-c b= | Buffers              | =ian/buffer-menu=      |
| =C-c d= | Dired                | =ian/dired-menu=       |
| =C-c e= | Editor operations    | —                      |
| =C-c f= | File utilities       | —                      |
| =C-c g= | AI / Git             | =ian/ai-menu=          |
| =C-c j= | Jump / Navigation    | =ian/jump-menu=        |
| =C-c l= | LSP (Eglot)          | — (eglot-mode-map)     |
| =C-c n= | Org-roam             | —                      |
| =C-c p= | Projectile           | =projectile-command-map= |
| =C-c q= | Quit / Utility       | —                      |
| =C-c s= | Search               | =ian/completion-menu=  |
