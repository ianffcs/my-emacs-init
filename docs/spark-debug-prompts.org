* Overview

Use these prompts in Open WebUI (https://oolong.lan) with ~qwen2.5-coder:32b~
to systematically debug and improve the modular Emacs configuration.

The config lives at =~/.emacs.d/= and is structured as:
- =init.el= — bootstraps and loads all modules in order
- =modules/core-*.el= — settings, packages, editor, UI, completion, auth, session
- =modules/tool-*.el= — dev tools, shell, dired, AI, comms, media, games
- =modules/lang-*.el= — language-specific configs (LSP, REPL, modes)
- =modules/ui-*.el= — navigation, windows, buffers, dashboard
- Package manager: =straight.el= + =use-package=

** 8.2 TRAMP SSH ControlMaster socket path

#+begin_example
My tool-dev.el sets TRAMP to use SSH ControlMaster with this path:

  -o ControlPath='~/.ssh/sockets/%%r@%%h-%%p'

But the directory ~/.ssh/sockets/ may not exist on spark. This silently makes
ControlMaster fail, causing each TRAMP operation to open a new SSH connection.

Write elisp that:
1. Creates ~/.ssh/sockets/ if it doesn't exist (with permissions 700)
2. Verifies the path works before setting it
3. Falls back gracefully if it can't be created
#+end_example

** 8.3 Clojure REPL MCP server conditional

#+begin_example
In my tool-ai.el, the MCP server list includes a conditional Clojure REPL entry:

  ,@(when (and (fboundp 'cider-current-repl)
               (ignore-errors (cider-current-repl)))
      `(("clojure" ...)))

This is evaluated once at config load time, before any REPL is started, so the
Clojure MCP server will NEVER be added. The condition will always be nil at init.

Rewrite this so the Clojure MCP entry is added dynamically when a CIDER REPL
connects (using cider-connected-hook) and removed when it disconnects
(using cider-disconnected-hook).
